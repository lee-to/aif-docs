# ai-factory Docs

Documentation site for the `lee-to/ai-factory` library.

Repository: https://github.com/lee-to/ai-factory

## Setup

Install dependencies:

```bash
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
bun run dev
```

## Production

Build the application for production:

```bash
bun run build
```

Locally preview production build:

```bash
bun run preview
```

Run checks:

```bash
bun run lint
bun run typecheck
```
