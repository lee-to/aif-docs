## Install

<code-group>

```bash [bun]
bun add -g ai-factory
```

```bash [pnpm]
pnpm add -g ai-factory
```

```bash [yarn]
yarn global add ai-factory
```

```bash [npm]
npm i -g ai-factory
```

```bash [mise]
mise use -g npm:ai-factory
```

</code-group>

## Quick start

<div class="code-group__hint">In your project directory</div>

```bash [Terminal]
ai-factory init
```

This will:

- Ask which AI agent you use
- Detect your project stack
- Install relevant skills
- Configure MCP servers (for supported agents)

From here you can plan and implement a feature.

## Your first project

### 1. Install AI Factory

<code-group>

```bash [bun]
bun add -g ai-factory
```

```bash [pnpm]
pnpm add -g ai-factory
```

```bash [yarn]
yarn global add ai-factory
```

```bash [npm]
npm i -g ai-factory
```

```bash [mise]
mise use -g npm:ai-factory
```

</code-group>

### 2. Go to your project

```bash [Terminal]
cd my-project
```

### 3. Initialize — pick one or more agents, detect stack, install skills

```bash [Terminal]
ai-factory init
```

Then open your AI agent and run:

```text
/aif
```

From there you can plan and implement a feature:

```text
/aif-plan Add user authentication with OAuth
/aif-implement
```

## Run without installation

<div class="code-group__hint">One-off execution</div>

<code-group>

```bash [bun]
bunx ai-factory init
```

```bash [pnpm]
pnpm dlx ai-factory init
```

```bash [yarn]
yarn dlx ai-factory init
```

```bash [npm]
npx ai-factory init
```

```bash [mise]
mise x npm:ai-factory -- ai-factory init
```

</code-group>
